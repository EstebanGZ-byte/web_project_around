# Tripleten web_project_around  

# Proyeto 7 JavaScript básico y trabajar con el DOM

Este proyecto es el número siete del programa de Desarrollo web en TripleTen que explica como poder lograr utilizar el lenguaje de programación JavaScript y enlazarlo a mi pagina web, este lenguaje ayuda en diferentes aspectos como el poder realizar operaciones aritmeticas y en el caso de esta pagina web hacer funciones y bucles; en este mismo proyecto se vio lo que es el DOM, o Modelo de Objetos del Documento, que es lo que le da a JavaScript acceso directo a los elementos de HTML y permite controlar el marcado sin editar el codigo HTML, quiere decir que trasnforma el contenido de una pagina web a un formato que JavaScript puede llegar a entender facilmente.
